# LZW-compression
Simple LZW compression algorithm using Python.

## To Encode a Text File
Open lzw-encoder.py and enter "filename.txt" which you want to encode
The file will be encoded in BIN format

## To Decode a BIN file
Open lzw-decoder.py and enter "filename.bin" which you want to decode
The decoded file will be saved in txt format

**Only works for txt to bin and vice versa**
